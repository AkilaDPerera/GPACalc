GPA Calculator Personal - Individual (2016)
This is a web application that helps students of University of Moratuwa, to calculate their semester GPA, current GPA and to keep the history of their grades.
How is that possible?
When the student enters his/her credentials for lms, this app will log on to his/her account and get the list of modules registered in each semester. After that app will show him/her a nice user-friendly UI to enter his/her grades received for each module. Then the app will do the calculation based on UoM performance criteria and display the results.
Features
  • User-friendly interface
  • Fault tolerant design
  • Reliable service
  • Daily DB backup
Technologies
  • Django python web framework
  • Jinja django templating language
  • Django ORM + SQLite DB
  • SMTP protocol client for mailling
  • Requests + BeautifulSoup in python for web-scrape
  • HTML5, CSS, JS
  • JQuery
  • Bootstrap

Tank Game UoM - Group of two (Nov 2016 – Dec 2016)
This is a multiplayer game where the players have to connect to a single game server and play. Game server communicates with each player by a set of pre-specified message set. According to the message received, players’ GUI will be updated. In this project, we were assigned to implement the player end with AI. Unity game engine and C# programming language were used to complete this project.
Technologies and tools
  • C# Programming Language
  • Unity3D Game Engine
  • Git version controlling
